Adress

O Elemento HTML address pode ser usado por autores para fornecer informação de contato para seu ancestral . Exemplo :
Voce pode contatar o autor em www.somedomain.com.
Se encontrar qualquer bug, por favor contate o administrador do site.
Você tambem pode querer nos visitar:
Mozilla Foundation
1981 Landings Drive
Building K
Mountain View, CA 94043-0801
USA

Blockquote

O Elemento HTML blockquote (ou Elemento HTML de citação de bloco) indica que o texto incluído é uma longa citação. Exemplo:

Esta é uma citação tirada da Mozilla Developer Center.

Code

O elemento HTML code representa um fragmento de código de computador. Por padrão, ele é exibido na fonte monospace padrão do navegador. Exemplo:

This is how we declare a Javascript variable: var i = 0;

DFN

O elemento HTML dfn> (ou Elemento Definição HTML) representa uma instância de definição de um termo. Exemplo:

A Internet é um sistema global de redes interconectadas que usam o Internet Protocol Suite (TCP/IP) para servir bilhões de usuários no mundo todo.

Span

O elemento HTML span é um conteiner generico em linha para conteúdo fraseado , que não representa nada por natureza. Ele pode ser usado para agrupar elementos para fins de estilo (usando os atributos class ou id ), ou para compartilhar valores de atributos como lang. Ele deve ser usado somente quando nenhum outro elemento semântico for apropriado. Exemplo:

Some text

Time

O elemento HTML time ( representa o tempo tanto no formato de 24 horas ou como uma data precisa no calendário Gregoriano (com informações opcionais de tempo e fuso horário) Este elemento é destinado a apresentar datas e horas no dispositivo em um formato legível. Isto pode ser útil aos agentes do usuário em oferecer qualquer programação de eventos ao calendário do utilizador. Exemplo: p>The concert starts at 20:00.
