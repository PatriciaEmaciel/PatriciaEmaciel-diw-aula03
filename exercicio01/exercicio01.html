Descrição das Tags HTML

Tag < em >

Esta tag é utilizada para dar enfase ao texto em uma pagina HTML. O seu comportamento padrão exibe o texto em italic.

Tag < Strong >

Esta tag é utilizada para dar importância ao texto. O seu comportamento padrão exibe o texto em negrito.

Tags < sup > e < sub >

Estas tags são utilizadas para definir texto sobrescrito e subescrito, respectivamente .São muito utilizadas em formulas quimicas e expressões matematicas com potencia.

Tags < ins > e < del >

Estas tags são utilizadas para definir inserções de texto e remoções de textos . Inserções e remoções são comuns em artigos e blogs,quando o autor quer deixar o usuario ciente de que houve uma atualização no texto,seja ela uma correção , inserção ou inclusão de palavras.O comportamento padrão deixa o texto sublinhado para inserções e tachado para exclusões.

Tag < Mark >

Esta tagé utilizada para marcar trechos do texto . É possivel marcar tanto palavras especificas quanto um conjunto de palavras . O comportamento padrão deixa o texto na cor preta e com o fundo amarelo.
